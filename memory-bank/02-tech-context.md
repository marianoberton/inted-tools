# Contexto Tecnológico

## Stack Principal
- **Framework**: Next.js 15.1.7
- **Lenguaje**: TypeScript
- **Estilos**: Tailwind CSS 3.4
- **UI Components**: Shadcn UI (basado en Radix UI)
- **Iconos**: Lucide React

## Librerías Clave
- **Utilidades**: `date-fns`, `uuid`, `clsx`, `tailwind-merge`
- **Generación de Documentos**: `html2pdf.js`, `numero-a-letras`
- **Procesamiento de Datos**: `xlsx` (procesamiento de Excel)
- **Animaciones**: `framer-motion`

## Scripts
- `npm run dev`: Inicia servidor de desarrollo (puerto 3002).
- `npm run build`: Construye la aplicación para producción.
- `npm run start`: Inicia la aplicación construida (puerto 3002).
- `npm run lint`: Ejecuta el linter.

## Configuración de Entorno
- Archivo `.env` para variables de entorno (no commitear secretos).
- Ejemplo en `.env.example`.

## Requisitos de Desarrollo
- Node.js (versión compatible con Next.js 15).
- Gestor de paquetes `npm`.
