# Project Brief: Inted Tools (inted-utils)

## Descripción General
Conjunto de herramientas internas y utilidades para la gestión de trámites y análisis comparativo.

## Objetivos
- Centralizar herramientas de uso cotidiano.
- Automatizar la generación de documentos (presupuestos).
- Proveer herramientas de comparación de ofertas.

## Usuarios
- Personal administrativo y técnico (e.g., del Estudio Minond).

## Alcance Actual (Activo)
- **Generador de Presupuestos**: Herramienta principal para crear presupuestos de trámites (ej. Prefactibilidad Urbanística) con generación de PDF.
- **Cuadro Comparativo**: Herramienta para comparar ofertas.

## Módulos Desactivados (Ocultos temporalmente)
- **Dashboard**: Análisis de procesos de compras.
- **Buscador**: Búsqueda general.
- **Boletín Oficial**: Consulta de boletines.

## No-Objetivos
- Por el momento no se define un sistema de autenticación complejo o SaaS público.
